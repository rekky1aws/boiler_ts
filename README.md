# TS

## Starting
Install all dependencies :
```bash
npm install
```

Start TS Compiler :
```bash
npm run start:build
# or
tsc -w
# or
npm run tsc -w
```

See code execution
```bash
npm run start:run
```

Install fs :
```bash
npm i @types/nodes
```
 -> Use this every time ther is an issue with a lib / package that doesn't have its types.
